编译Wenet runtime

需要有cmake、g++

在wenet/runtime/server/x86 进行：mkdir build && cd build && cmake .. && cmake –build .

调试wenet

需要有gdb（安装的gdb 9.2，./configure –prefix=… && make -j all && make install）（安装gdb需要有texinfo（安装的texinfo 6.6，./configure –prefix=… && make -j all && make install））
